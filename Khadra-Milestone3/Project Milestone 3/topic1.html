What Is HTTP?

HTTP is the protocol that allows browsers and servers to communicate. Whenever you visit a website, your browser sends a request to a server, and the server responds with content such as HTML, images, scripts, or data. This simple model forms the basis of nearly everything that happens on the web.

How the Request–Response Cycle Works

  • Request: The browser asks the server for a resource.
  • Response: The server returns a status code, headers, and the requested content.
  • Resources: These include HTML, CSS, JavaScript, images, JSON API data, and more.

This model has remained the same since the first version of HTTP, even as the protocol expanded to support new features and modern performance needs.
